# SafeMind GitHub Pages

This directory contains the GitHub Pages site for SafeMind.

## View the Site

Visit: https://allisonmh.github.io/safemind-draft1

## Local Development

To run locally:

```bash
# Serve with Python
cd docs
python3 -m http.server 8000
```

Then open http://localhost:8000

## Structure

- `index.html` - Main landing page
- `styles.css` - All styling
- `script.js` - Interactive features
- `_config.yml` - Jekyll configuration (for GitHub Pages)

## Deployment

GitHub Pages automatically deploys from the `/docs` folder on the main branch.

To enable:
1. Go to repository Settings
2. Navigate to Pages
3. Select "Deploy from a branch"
4. Choose `main` branch and `/docs` folder
5. Save

The site will be live at: https://allisonmh.github.io/safemind-draft1

## Customization

Edit the HTML, CSS, and JS files directly. Changes will be reflected after pushing to GitHub.
