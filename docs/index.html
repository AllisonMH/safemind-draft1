How It Works

A comprehensive approach to youth safety in AI conversations

1

Content Capture

AI conversation messages are captured and sent for analysis through secure APIs

‚Üí
2

Multi-Layer Analysis

Text runs through multiple detection layers simultaneously:

  • Perspective API (toxicity, threats)
  • OpenAI Moderation (self-harm, violence)
  • Keyword Detection (mental health)
  • Sentiment Analysis

‚Üí
3

Risk Assessment

Results are weighted and combined into a comprehensive risk score (0-100) with categorization:

  • Low (0-40) - Safe content
  • Medium (40-60) - Monitor
  • High (60-80) - Concerning
  • Critical (80-100) - Immediate action

‚Üí
4

Action & Alert

Based on risk level, the system:

  • Generates suggested actions
  • Alerts parents/guardians if needed
  • Provides crisis resources
  • Logs for pattern analysis
